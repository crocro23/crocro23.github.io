🌐 Comprendre Internet et ses services

Internet est un réseau mondial permettant aux ordinateurs de communiquer entre eux.
Il propose plusieurs services essentiels :

Services classiques

  • Le Web : le plus connu, utilisé pour consulter des sites web
  • Les e-mails : échanges de messages
  • Les newsgroups : forums de discussion historiques
  • Le FTP : transfert de fichiers

Services cloud modernes

  • Messagerie électronique : Gmail, Outlook
  • Stockage en ligne : Dropbox, Google Drive
  • Messagerie instantanée : Slack, Discord, Messenger

🧱 Les langages du Web (côté client)

Les langages « client » s’exécutent dans le navigateur et permettent d’afficher et d’animer un site web.

HTML — structure du contenu

Définit les textes, images, liens, titres…
Exemple interactif :
https://codepen.io/OpenClassroomsLearning/pen/WNvKdbp?editors=1000

CSS — mise en forme

Gère les couleurs, polices, tailles, marges, animations…

JavaScript — interactions

Permet d’ajouter du dynamisme : menus déroulants, formulaires interactifs, animations…

📱 Rendre un site compatible smartphone

Deux approches existent :

1. Le site web responsive

Le site s’adapte automatiquement à la taille de l’écran.
Technologies utilisées : HTML, CSS, JavaScript.

2. L’application mobile native

Développée spécifiquement pour un smartphone.

  • Android : Kotlin, Java
  • iOS : Swift, Objective‑C

Plus coûteux mais idéal pour des services complexes.

Les langages backend définissent le comportement du site ou de l’application.
